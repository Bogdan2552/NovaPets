
### 1. Install Node.js

Install Node.js LTS from the official Node.js website. After installation, check that Node and npm are available:

```bash
node -v
npm -v
```

If both commands show version numbers, the setup is ready.

### 2. Open the project folder

Unzip the project archive and open the folder in VS Code or another editor.

In the terminal, move into the project folder:

```bash
cd NovaPets-React-Assignment
```

### 3. Install dependencies

Run:

```bash
npm install
```

This installs React, React DOM, Vite, and React Router DOM from `package.json`.

### 4. Start the development server

Run:

```bash
npm run dev
```

Vite will print a local URL, usually:

```txt
http://localhost:5173/
```

Open that URL in the browser.
